# 🚀 Portafolio Personal - Silvio Barrios

¡Bienvenido a mi portafolio! Soy Silvio Barrios, un apasionado **Desarrollador Web** en formación en **Riwi**. Este proyecto es una vitrina de mis habilidades técnicas, mis proyectos actuales y un espacio personal para conocer a mi "equipo" de mascotas.

## 🛠️ Tecnologías Utilizadas

*   **HTML5:** Estructura semántica avanzada.
*   **CSS3:** Diseño responsivo, Grid, Flexbox y animaciones.
*   **JavaScript (Vanilla):** Manipulación dinámica del DOM y lógica de menús.
*   **Diseño:** Enfoque en accesibilidad y UX/UI oscuro.

## 📂 Secciones del Proyecto

1.  **Sobre mí:** Un resumen de mi trayectoria y enfoque como Coder.
2.  **Proyectos:** Lista detallada de retos técnicos (Riwi Bank, Gestores, etc.).
3.  **Contacto:** Formulario funcional para networking.
4.  **Mis Mascotas:** Una sección interactiva desarrollada con **CSS Grid** para mostrar a mis compañeros de vida.

## 🔧 Instalación y Visualización

Si deseas ver este proyecto de forma local:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/silvio1615/Portafolio.git
   ```
2. Abre el archivo `index.html` en tu navegador.
